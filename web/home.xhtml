Quick start

How to run a job


  • Private or public way

    • Go to
    • Create new project selecting either public or private visibility
    • Select newly created project
    • Upload experiment datasets
    • Select and run job from menu on the left
    • When job finishes, new entries will appear in the project datasets
    • Alternatively you can view specific job outputs by clicking its name on Workflow History

  • Workgroup way

    • Go to
    • Create new workgroup
    • Select newly created workgroup
    • Create new project in this workgroup
    • Select newly created project
    • Upload experiment datasets
    • Select and run job from menu on the left
    • When job finishes, new entries will appear in the project datasets
    • Alternatively you can view specific job outputs by clicking its name on Workflow History

Workgroups


Workgroups allow you to organise your work with other people.

All members of a workgroup are able to see and make their input to any projects under given workgroup.

To view and manage your workgroups visit


Projects


Projects help you to organise and keep track of your experiments jobs.

To run a job, first you need to create and select a project.

Types of projects:

  • Private – visible only to its owner
  • Protected – visible to its assigned workgroup
  • Public – visible to all users

To view and manage your projects visit
